📋 دليل الاستخدام التفصيلي

🖱️ التحكم بالفأرة

  • عجلة للأعلى = تكبير زمني
  • عجلة للأسفل = تصغير زمني
  • Ctrl + عجلة = تكبير سعري
  • سحب = تحريك الرسم البياني
  • نقر مزدوج = تكبير سريع

⌨️ اختصارات المفاتيح

  • ESC = إعادة تعيين شامل
  • + = تكبير
  • - = تصغير
  • مفاتيح الأسهم = التنقل
  • Ctrl+0 = إعادة تعيين

🎛️ أزرار التحكم

  • 🔍+ = تكبير زمني
  • 🔍- = تصغير زمني
  • 📈 = تكبير سعري
  • 📉 = تصغير سعري
  • 🔄 = إعادة تعيين
