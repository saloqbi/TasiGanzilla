🛠️ خطة الحل النهائية

الخطوة 1: إيقاف جميع العمليات

# في Task Manager ابحث عن: - node.exe - npm.exe - vite # وأوقف جميع العمليات

الخطوة 2: مسح node_modules و cache

rmdir /s "node_modules" rmdir /s ".vite" npm install

الخطوة 3: بدء تشغيل نظيف

npm run dev

الخطوة 4: اختبار المكون البسيط

أولاً جرب المكون البسيط، ثم ارجع للمكون الكامل
