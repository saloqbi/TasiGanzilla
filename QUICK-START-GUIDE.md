# 🎯 دليل التشغيل السريع - نظام التحليل المتقدم مع MongoDB

## ✅ النظام مكتمل ويعمل!

### 🚀 كيفية التشغيل

#### الطريقة الأولى: تشغيل كامل بنقرة واحدة
```batch
# انقر مرتين على الملف
run-complete-system.bat
```

#### الطريقة الثانية: تشغيل يدوي متقدم
```powershell
# 1. تشغيل خادم MongoDB
cd server
node server.js

# 2. تشغيل تطبيق React (في نافذة أخرى)
npm run dev

# 3. فتح المتصفح
# http://localhost:3001
```

### 📋 كيفية الاستخدام

1. **افتح التطبيق**: استخدم `run-complete-system.bat`
2. **تحقق من الاتصال**: ابحث عن 🟢 في قسم "حالة قاعدة البيانات"
3. **استعرض البيانات**: انقر على "عرض قاعدة البيانات" في الشريط العلوي
4. **فعّل البيانات الحقيقية**: استخدم مفتاح "البيانات الحقيقية" في الإعدادات
5. **ابدأ التحليل**: اختر الرمز والإطار الزمني وابدأ التحليل
